#  Calculadora em VueJS

Projeto simples que simula calculadora simples durante curso de VueJS da COD3R .

Bibliotecas e tecnologias utilizadas:
- VueJS

![Calc_VueJS](https://i.imgur.com/bv2LBEk.png"Calc_VueJS")

### Instalação

requisitos

- Node instalado

### Passo a Passo

-  clone o repositório https://github.com/AndersonBarbosadaCosta/VueJS-Calculadora.git

- abra a pasta do projeto em seu editor preferido

-  instale as dependências do projeto utilizando **npm** ou **yarn**

    executar o comando  npm install  ou yarn install
    subir a aplicação  npm run serve

Podemos conferir a aplicação no endereço: http://localhost:8080
